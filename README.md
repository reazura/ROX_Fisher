# ROX_Fisher

Requirements: 

    - 1080p resolution, 

    - LDPlayer

    - pip install the following packages: 
        pyautogui,opencv,mss,keyboard

Setup:
    
    Toggle LDPlayer in Mini Mode, persist to be always on top then adjust "max" variable to how much should be fished in a session.
